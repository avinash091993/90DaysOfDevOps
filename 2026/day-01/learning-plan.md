# 90 Days DevOps Learning Plan

## My Understanding of DevOps & Cloud

DevOps is a practice that focuses on automating software delivery and improving collaboration between development and operations teams. It helps in faster, reliable deployments using tools like CI/CD, Docker, and Kubernetes.

Cloud Engineering involves using platforms like AWS to create scalable, highly available infrastructure without managing physical servers.

- To build strong real-world infrastructure and automation skills
- To become job-ready in a short time (45–90 days focus)
